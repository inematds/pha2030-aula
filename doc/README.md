# PHA 2030

Materiais-base do curso, palestra e workshop "Da capacitação à transformação: como preparar pessoas e empresas para a era da IA".

## Estrutura

- `conteudo-mestre.md`: versao principal da aula, com narrativa completa.
- `apostila-aluno.md`: material didatico detalhado para estudo individual.
- `slides.md`: versao resumida para apresentacao em slides.
- `apresentacao-completa.md`: apresentacao longa com narrativa por slide.
- `apresentacao-executiva.md`: apresentacao curta para reunioes e pitch.
- `roteiro-palestra.md`: roteiro com falas prontas do apresentador.
- `workshop-2h.md`: versao de workshop com dinamica, exercicios e conducao.

## Uso recomendado

1. Use `conteudo-mestre.md` como documento central de autoria.
2. Use `apostila-aluno.md` como base de estudo e material de apoio.
3. Adapte `apresentacao-completa.md` para PowerPoint, Keynote, Canva ou Gamma.
4. Use `apresentacao-executiva.md` em reunioes com empresas, liderancas e parceiros.
5. Use `roteiro-palestra.md` para apresentacao ao vivo.
6. Use `workshop-2h.md` em turmas corporativas ou experiencias mais praticas.
